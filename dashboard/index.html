Triviu · Public dashboard

Every execution, failures included

A metric without its failure context is marketing; a metric with failures included is evidence. This dashboard publishes every execution — the reverts and the losses alongside the wins — because that is the only honest way to show what atomic arbitrage actually does.

RISK NOTICE — REQUIRED READING

Most atomic-arbitrage opportunities on Polygon are captured by professional operators. For most individual users, the expected result after gas and fee tends toward zero or negative. Triviu is educational infrastructure — not a source of income. Reverted transactions still pay gas.

The dashboard goes live with the first deployment

There is no on-chain deployment yet, so there is nothing honest to plot — and Triviu does not invent numbers. The moment the Executor is deployed and verified, this dashboard fills in from public on-chain data, with failures included from the first block.

The queries that will feed it are already public and versioned: revert rate, executions by route, gas vs net result. Don't trust — verify: read the SQL yourself.
