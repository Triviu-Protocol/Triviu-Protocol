The public record, failures included.

A metric without its failure context is marketing; a metric with failures included is evidence. This dashboard will publish every execution — the reverts and the losses alongside the wins — because that is the only honest way to show what atomic arbitrage actually does.

Where this stands

Nothing is deployed on any chain yet, so there is nothing honest to plot — and Triviu invents no numbers. The moment the Executor is deployed and verified, this dashboard fills in from public on-chain data, per chain, with failures included from the first block. Reverted transactions still pay gas.
